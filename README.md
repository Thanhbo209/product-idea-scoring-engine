# AI-Powered Idea Validator

A fullstack platform for validating product ideas using AI-driven scoring, versioning, and structured feedback analysis.

## Overview

AI-Powered Idea Validator helps users evaluate early-stage product ideas by providing:

- AI-based idea scoring
- Version tracking for iterative improvements
- Structured feedback generation
- Comparison between idea versions
- Evaluation history for decision-making

The system is designed to reduce uncertainty in early product development and support data-driven validation of ideas before implementation.

---

## Features

### Core Features
- Submit product ideas with title and description
- AI evaluation of idea quality (clarity, feasibility, impact, originality)
- Version control for each idea
- Compare multiple versions of the same idea
- Evaluation history tracking

### AI Integration
- Automated scoring using LLM (Gemini / OpenAI)
- Structured feedback generation
- Improvement suggestions per version
